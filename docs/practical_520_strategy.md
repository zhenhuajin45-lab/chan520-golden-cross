# 520实战版策略说明

本文件把材料第三章中与 520 相关的内容，整理成本地可执行、可复盘的策略规则。目标不是把所有交易方法混合在一起，而是把 520 定位为“中短线趋势/波段模式”，并明确适用边界、买点、持仓、加仓和退出。

## 1. 从简易版到实战版

简易版只看三件事：

- 股价站上 MA5 和 MA20。
- MA5 上穿 MA20。
- 最好放量，并配合 MACD 或量能。

实战版必须补齐六个环节：

| 环节 | 本地规则 |
| --- | --- |
| 模式选择 | 只做趋势向上、平台放量突破、底部放量反弹三类形态；不把龙头打板、红利防御和520混成一个系统 |
| 大周期过滤 | 60日涨幅、MA60、周线MA60、MA250、均线多头/修复状态 |
| 520买点 | 当天金叉、近3日确认、股价在MA5/MA20上方、量能、MACD、短线偏离 |
| 趋势5日线 | 主升段回踩5日线不破、缩量止跌、阳线收回、MACD修复，满足5项中至少3项 |
| 加仓位置 | 平台突破后缩量回踩不破，或回踩5/10/20日线止跌再放量；采用金字塔加仓 |
| 失败退出 | MA5放量跌破、MA20跌破、520死叉、高位天量滞涨、长上影、短期涨幅过快 |

## 2. 适用边界

520适合几天到几周的中短线波段，不适合 T+0、长期下跌趋势、剧烈波动的妖股或纯题材情绪打板。龙头战法可以作为市场情绪参考，但不是本策略的买入核心；红利周期策略属于防御/资产配置逻辑，也不并入 520 打分。

实战执行时优先做一套模式：做520，就按趋势/波段规则；没有信号就空仓等待。

## 3. 标准520入选

标准入选需要同时满足：

- MA5 当天上穿 MA20。
- 收盘价在 MA5 和 MA20 上方。
- 成交量温和放大，默认近5日量比大于等于 1.2。
- MACD DIF > DEA，且 DIF/DEA 都在零轴上方更强。
- 大周期不是下跌趋势，最好 60日涨幅大于等于 5%。
- 没有明显追高：收盘相对 MA5 偏离小于 8%，相对 MA20 偏离小于 15%，RSI 不过热。

若金叉成立但 MACD 还在零轴下方、大周期只是修复、均线多头未完成，判为“弱化520”，只给观察或轻仓试探。

## 4. 近3日确认与追高过滤

520金叉不是只有当天有意义。金叉后 2-3 日如果继续站在 MA5/MA20 上方、量价配合、没有跌回关键均线，可以视为确认期。

但确认期不等于继续追买。若出现涨停、快速远离 MA5/MA20、RSI过热或短期涨幅过快，空仓者应等待回踩；已持仓者按5日线持有。

## 5. 趋势行情5日线规则

趋势5日线买入法的前提是：个股已经完成筑底或修复，并突破 MA60，MA5 强于 MA20。

满足以下5项中的至少3项，才算趋势5日线买入/持有结构：

- 5日线上穿10日、20日线，或短期均线多头发散。
- 放量上攻后首次回踩5日线不破。
- 回踩或横盘过程中缩量止跌。
- 回踩5日线后阳线收回，且成交量温和放大。
- MACD绿柱缩短或红柱继续增强。

持股规则：

- 收盘在5日线上方，持股不动。
- 回踩5日线但收回，且没有放量下跌，继续持有。
- 长上影但未破5日线，可以减仓但不必清仓。
- 放量跌破5日线，先减仓；反抽不过5日线，再退出。

## 6. 加仓规则

只顺势加仓，不逆势补仓。

可考虑加仓的位置：

- 首次突破平台或前高后，缩量回踩不破。
- 上升趋势中回踩5日、10日或20日线附近止跌。
- 出现“放量阳线 + 缩量回踩 + 再次放量收回”的组合。

禁止加仓的位置：

- 跌破 MA20/MA60 后为摊低成本而补仓。
- 涨停或大阳线后已经明显远离 MA5。
- 没有预设止损、满仓倒金字塔加仓。

## 7. 退出规则

退出优先级高于买入信号：

- MA5 下穿 MA20，或股价跌破 MA20 后反抽不回。
- 放量跌破 MA5，尤其伴随大阴线。
- MACD死叉或柱状图持续转弱。
- 高位天量滞涨、长上影、十字星，或短期涨幅超过20%-30%后剧烈波动。
- 买入后 2-3 日没有按预期上攻，反而横盘走弱。

本地报告会把这些列在“退出与止盈风险”里；如果退出风险触发，即使前面有买点，也会降级为回避/减仓观察。

## 8. 指标口径

默认 MACD 使用常见的 `12,26,9`。若需要和同花顺截图核对，可运行：

```powershell
python -m chan520_skill analyze 600288 --date 2026-07-07 --macd-slow 50
```

同花顺截图里的 `MACD(12,50,9)` 与默认 `12,26,9` 数值不同，结论应以同一参数口径比较。
