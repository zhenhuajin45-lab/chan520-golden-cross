# 交易框架蒸馏说明

本地 skill 的目标不是复述交易心法，而是把框架中可计算、可复盘、可执行的部分落成规则。

## 1. 框架核心

- 趋势优先：方向大于技巧，不与大周期趋势为敌。
- 少交易：无机会时空仓，避免震荡期频繁交易。
- 买点必须有结构：单一金叉不够，需要均线、量价、MACD、支撑位共同验证。
- 风险先于收益：每个买点都必须带止损、仓位和确认失败条件。
- 分层处理信号：标准信号、弱化信号、观察信号、未触发要分开。

## 2. 三周期映射

| 框架语言 | 本地实现 |
| --- | --- |
| 大周期看方向 | 60日涨幅、MA60、MA250、周线MA60 |
| 中周期看结构 | MA5/MA10/MA20/MA60 排列、520金叉、MACD |
| 小周期抓节奏 | 量比、MA5斜率、突破MA60、RSI |

## 3. 520 战法规则

标准 520 金叉买：

- MA5 今日上穿 MA20。
- 收盘价站上 MA5 和 MA20。
- 成交量放大。
- MACD 同步金叉，最好在零轴上方。
- 大周期满足上升过滤。

弱化 520 金叉买：

- MA5 上穿 MA20 成立。
- MACD 同步金叉但仍在零轴下方。
- 大周期 60日涨幅不足 5%，或均线多头排列未完成。
- 只能给出观察/轻仓试探，不升级为标准入选。

卖出/失败：

- 跌破 MA20 或 MA60 且放量。
- MA5 下穿 MA20。
- MACD 死叉或柱状图转弱。
- 金叉后 2-3 日不能站稳关键均线。

## 4. 缠论买点近似

原始缠论需要人工划分分型、笔、线段和中枢。为了本地可执行，当前实现采用近似规则：

- 一买：阶段新低后 MACD 下行动能不再同步创新低，视作左侧背驰试错。
- 二买/三买：突破中枢代理线（当前用 MA60/平台突破代理）后，等待回踩不破确认。
- 突破回踩买：站上 MA60 后不追高，优先等 2-3 日确认或回踩不破。

这些近似规则的作用是辅助报告分层，不替代人工严格缠论画线。

## 5. 截图能力对照

已实现的截图式报告能力：

- 标题、数据截止日、收盘价。
- 三周期扫描结论。
- 选股判定表。
- 买卖点判定表。
- 520 金叉逐项拆解。
- 满足条件与主要缺陷分列。
- 空仓者、已持仓者、理想买点、止损位、目标位。
- 核心结论使用“弱化但有所改善”“观察/轻仓试探”“未来2-3日确认”等框架语言。
